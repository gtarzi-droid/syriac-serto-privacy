Enabling the keyboard

  1. Open Settings › System › Languages & input › On-screen keyboard › Manage keyboards.
  2. Turn on “Syriac Serto” and confirm the security prompt.
  3. In any text field, tap the keyboard-switch button (bottom-right) or the globe key (🌐) and pick Syriac Serto.

Typing

The layout is phonetic and QWERTY-mapped: each Syriac letter sits at the Latin key with the nearest sound. The u, i, and o keys carry the West-Syriac Serto vowels. Cursive shaping (initial/medial/final letterforms) is automatic. Press and hold a key for related letters and variants.

A note on how it looks in other apps

Everything you type is real Unicode Syriac. Inside this app (keycaps and preview) it is drawn in a Serto face. Other apps draw Syriac in the system's own Syriac font, which may look different — Android does not allow any app to change the system font.
